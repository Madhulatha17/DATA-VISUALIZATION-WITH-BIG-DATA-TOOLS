## Overview
This project visualizes sales data using BI tools like Power BI / Tableau / Looker Studio.



## Objective
To analyze and visualize sales, profit, region, and product performance.


##  Dataset
The dataset contains:
- Order ID
- Product
- Category
- Region
- Sales
- Profit
- Date


##  Dashboards Created
- Sales by Category
- Profit by Region
- Product Performance
- KPI Scorecards



##  Output
Dashboard screenshot available in:
dashboard_screenshots/


##  Tools Used
- Google Sheets
- Looker Studio / Power BI / Tableau


##  Key Insights
- Electronics is the top-performing category
- South region has highest sales
- Laptop is the best-selling product
- Profit varies across categories and regions
